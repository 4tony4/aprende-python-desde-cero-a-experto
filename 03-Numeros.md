# Números
Python soporta dos tipos de números: enteros (integer) y de punto flotante (float)
 
    # integer
    x = 5
    print(x)
    
    # float
    y = 5.0
    print(y)
    
    # Otra forma de declarar un float
    z = float(5)
    print(z)

Si tenemos dudas del valor de una variable, podemos utilizar la función `type()`:

    >>> x = 5.5
    >>> type(x)
    <class 'float'>
